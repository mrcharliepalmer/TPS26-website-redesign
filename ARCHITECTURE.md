## Technical Requirements

- Single HTML file per page, shared styles.css
- Vanilla JS only (nav toggle, countdown, smooth scroll)
- Mobile-first responsive
- Every page: title, meta description, OG tags
- Homepage: JSON-LD structured data (Event schema)
- FAQ page: FAQ schema markup
- Fast loading — optimised images, no frameworks
